## ¿Qué es AlternativaSoberana?

AlternativaSoberana es una plataforma web desarrollada por Xyon Platforms. Construida con TypeScript y Vite, con despliegue self-hosted vía Docker y Coolify.

## ✨ Funcionalidades

- Interfaz web moderna y reactiva
- Gestión y administración de contenido
- Despliegue self-hosted vía Docker

## 🛠️ Stack técnico

| Capa | Tecnología |
|------|----------|
| Frontend | React + TypeScript + Vite |
| Backend | Node.js + TypeScript |
| Despliegue | Docker + Coolify |

## 🚀 Instalación

```bash
git clone https://github.com/mkandreum/AlternativaSoberana.git
cd AlternativaSoberana
cp .env.example .env
docker-compose up -d
```

### Desarrollo local

```bash
npm install
npm run dev
```
